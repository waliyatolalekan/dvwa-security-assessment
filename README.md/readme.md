This project demonstrates the complete lifecycle of web application security assessment in a controlled virtual environment. It covers reconnaissance, vulnerability discovery, controlled exploitation, digital forensic analysis, SSL/TLS implementation, server hardening, and post-hardening validation using Kali Linux, Metasploitable 2, and DVWA.

Objectives
Perform network reconnaissance
Discover web application vulnerabilities
Conduct controlled exploitation
Analyze attack traces
Implement HTTPS
Harden the server
Reassess the environment

Tools Used
Kali Linux
Metasploitable 2
DVWA
Nmap
Nikto
Dirb
Metasploit Framework
Wireshark
OpenSSL
Apache2

This project was conducted entirely in an isolated virtual lab environment using intentionally vulnerable systems. No testing was performed against systems without authorization.
